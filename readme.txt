You can easily deploy this repository to your own cloud, linux, or other host running php, and MySQL. See, howmanycupsofcoffee.com for an example.

Step 1) Create a new directory in the web directory on your server, and extract the repository into it.
Step 2) Find the "coffee.sql" file, and use it to create a new database. Update the config.php file with this info.
Step 3) Find the "config.php" file in the "middlelayer" directory. Edit your settings, and save the file.
Step 4) Visit yourdomain.com/yourDirectoryName. Please note: The site is was designed to work on small screens. You may find it usefull to bookmark the site as an app icon on your phone.

howmanycupsofcoffee.com - this is the website code

You can easily deploy this repository to your own cloud, linux, or other host running php, and MySQL.

Step 1) Create a directory on your server, and extract the repository into it
Step 2) Find the "config.php" file in the "middlelayer" directory. Edit your settings and save the file.
Step 3) Find the ****.sql file, and use it to create a new database which should reference in the config.php
Step 4) You're done! Grab a cup of coffee.

INSTRUCTIONS -----------------------------------------------------------------------------------------------------------------------------------------------------------------

You can easily deploy this repository to a cloud, linux, or other host. Provided it's running php, and MySQL. I'm running mine at howmanycupsofcoffee.com.

Step 1) Create a new directory in the web directory on your server, name it, and extract the repository into it.
Step 2) Find the "coffee.sql" file in the root directory, and use it to create a new database.
Step 3) Find the "config.php" file in the "middlelayer" directory. Edit your settings, and save the file.
Step 4) Visit yourdomain.com/yourDirectoryName. Please note: The site is designed to work on mobile devices. You may find it usefull to bookmark the site as an app icon on your phone.

Enjoy!

howmanycupsofcoffee is being actively developed, check for updates often. To upgrade: save your config.php file someplace, replace all the other files and folders with the new content you download from https://github.com/jasonhejna/howmanycupsofcoffee.
